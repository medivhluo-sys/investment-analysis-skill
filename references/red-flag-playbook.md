# 红旗剧本

红旗不是泛泛风险，而是会改变是否推进、估值、条款或后续资料要求的事实或不确定性。每个红旗必须绑定证据等级、需要补充的资料和个人投资者动作。

## 严重性

| 等级 | 含义 | 个人投资者动作 |
|------|------|----------------|
| P0 | 投资逻辑可能失效或交易不可执行 | 暂停、拒绝或要求完全修复后再看 |
| P1 | 需要补证、降价、加条款或设置前置条件 | 有条件关注，补资料或重算估值 |
| P2 | 可跟踪管理的问题 | 纳入投后关注和预警清单 |

## 红旗卡片

| 红旗 | 触发信号 | 查验动作 | 投资者动作 |
|------|----------|----------|------------|
| 收入真实性存疑 | 收入无法对应合同、交付、验收、发票或回款 | 抽样穿透前五大客户，做客户访谈和公开客户线索核验 | P0/P1；暂缓正面判断 |
| 毛利率异常 | 毛利显著高于同行或历史，但没有产品、客户或成本解释 | 拆产品/客户毛利，对比原材料、同行和可比公司 | P1；用调整后利润估值 |
| 业务包装 | 贸易、集成或渠道业务被包装成高科技、平台或制造 | 查人员、研发、IP、生产、客户和供应商链条 | P1/P0；重分类估值方法 |
| 关联交易依赖 | 销售、采购、资金、资产或人员与关联方重叠 | 做关联方图谱，查定价、公允性和可替代性 | P1；要求披露、清理或治理保护 |
| 客户集中 | 少数客户决定收入、毛利或回款 | 查合同期限、续约、回款、客户访谈和替代客户 | P1；要求客户确认和安全边际 |
| 供应商集中 | 关键原材料、产能或技术依赖单一供应商 | 查替代供应商、交期、价格传导和合同约束 | P1/P2；设置供应风险条件 |
| 回购依赖 | 收益主要来自创始人、大股东或关联方回购 | 查回购方资产、现金流、负债、诉讼和担保 | P1/P0；按信用风险处理 |
| 退出不可验证 | IPO、并购或转让只有愿景，没有案例或路径 | 查上市条件、可比退出、潜在买方和锁定期 | P1；不得输出确定收益率 |
| 现金转化差 | 利润增长但应收、库存、合同资产增长更快 | 查账龄、后续回款、营运资金和坏账 | P1；下调估值和现金流 |
| 实控人诚信疑点 | 陈述反复、隐瞒资料、重大诉讼或关联方异常 | 交叉核验访谈、公开记录和交易资料 | P0；诚信问题优先级高于估值 |
| 资质或合规缺口 | 许可证、环保、安全、数据、药监等前置条件缺失 | 查主管部门、公示、批文和处罚 | P0/P1；未修复前不推进 |

## 写作规则

- 不写“市场竞争风险”这类通用风险，必须写公司特定机制。
- 已经发生的问题写“问题”或“影响”，不要写成风险。
- P0/P1 必须有补资料清单或交易动作。
- 无法量化时，至少说明影响收入、利润、现金流、估值、退出还是执行。
