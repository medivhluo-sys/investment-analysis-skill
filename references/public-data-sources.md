# 公开数据源发现协议

本文件不是免费数据源穷举清单。投资标的行业、地域和数据口径变化很快，清单无法覆盖所有场景。它的作用是提供：

1. 常用权威入口的种子库
2. 新行业数据源发现方法
3. 来源可靠性分级
4. Source Card 记录模板
5. 找不到免费权威来源时的处理规则

## 核心原则

- 官方优先：监管、政府、交易所、法院、工商、知识产权、统计、海关、税务、行业主管部门优先。
- 原始披露优先：公告、招股书、年报、债券募集说明书、审计报告、行政文书优先于二手转载。
- 可追溯优先：必须能回到 URL、文件名、页码、发布时间、发布机构。
- 口径一致优先：币种、税前税后、含/不含 VAT、期间、统计范围必须可解释。
- 免费不等于充分：免费公开来源不足时，必须提示线下尽调、付费数据库或第三方核验路径。

## 来源可靠性分级

| 等级 | 来源类型 | 可支持的结论 |
|------|----------|--------------|
| S | 政府、监管、法院、交易所、工商、知识产权、统计局、海关、行业主管部门等官方原始来源 | 可作为关键事实主证据 |
| A | 上市公司公告、招股书、债券募集说明书、审计报告、评级报告、正式法律/财务/税务 DD 报告 | 可作为财务、估值、交易和可比公司主证据 |
| B | 行业协会、国际组织、学术论文、头部券商/咨询/评级机构研究报告 | 可支持行业趋势、市场空间和对标假设，但需交叉验证 |
| C | 企业官网、新闻报道、访谈纪要、FA/GP 材料、招聘信息、公众号 | 只能作为线索或辅助证据 |
| D | 自媒体、论坛、无作者/无日期/无法追溯转载、AI 摘要 | 不得作为投资结论依据 |

关键结论至少需要 1 个 S/A 来源，或 2 个相互独立的 B 来源。只有 C/D 来源时，结论必须标注“待验证”。

## 数据源发现流程

每次新项目先按以下维度拆解数据需求：

1. 地域：境内、境外、跨境、多地运营
2. 主体：非上市公司、上市可比公司、项目公司、SPV、基金、实控人
3. 行业：药品、医疗器械、能源、化工、汽车、半导体、教育、金融、数据安全、环保、安全生产等
4. 数据类型：工商、诉讼、知识产权、财务、价格、产能、进出口、招投标、环保、安全生产、许可资质、政策补贴
5. 决策用途：验证收入、成本、市场空间、竞争格局、治理风险、退出估值、CP 条件

然后按以下顺序发现来源：

1. 行业主管部门和监管机构
2. 国家/地方统计部门、海关、税务、发改、工信、商务、药监、能源、生态环境、应急管理等
3. 交易所、上市公司公告、招股书、债券募集说明书
4. 法院、工商、知识产权、招投标、行政处罚、信用信息系统
5. 行业协会、标准组织、国际组织
6. 头部券商、评级、咨询、期刊论文
7. 企业官网、新闻、访谈、FA 材料

## 未知行业模式

当标的行业不在种子库中：

1. 识别行业主管部门和核心监管事项。
2. 搜索该行业是否存在许可证、认证、强制标准、行政处罚、环保/安全生产要求。
3. 找 2-3 家上市或发债可比公司，读取公告、招股书、年报或募集说明书。
4. 搜索国家标准、行业标准、地方规划、招投标记录、协会统计。
5. 为每个新来源建立 Source Card，并在报告附录说明为什么采用。
6. 不要把一次项目中发现的新来源永久写入 skill，除非用户明确要求沉淀。

## Source Card 模板

每个新增数据源必须记录：

| 字段 | 内容 |
|------|------|
| 来源名称 | 官方名称或文件名称 |
| URL / 文件路径 | 可追溯链接或本地文件路径 |
| 发布/主管机构 | 机构全称 |
| 数据类型 | 工商、财务、价格、诉讼、许可、统计等 |
| 覆盖范围 | 地域、行业、主体、时间区间 |
| 更新时间 | 发布时间或访问日期；动态网页写访问日期 |
| 是否免费 | 免费、需账号、需 API key、付费、待确认 |
| 可验证字段 | 本次能验证哪些事实或假设 |
| 局限性 | 滞后、口径不一致、样本不足、非官方等 |
| 本次用途 | 支撑哪条结论或假设 |
| 证据等级 | S/A/B/C/D |

## 常用种子入口

| 场景 | 来源 | 用途 | 备注 |
|------|------|------|------|
| 中国企业工商、行政处罚、年报公示 | 国家企业信用信息公示系统 / 市场监管总局相关服务 | 主体存续、股东、处罚、年报线索 | 官方入口优先；网页字段可能需要人工核验 |
| 中国上市公司和可比公司 | 巨潮资讯、上交所、深交所、北交所、港交所 | 公告、招股书、年报、并购、问询函 | 以原始公告 PDF 为准 |
| 中国宏观和行业 | 国家统计局、地方统计局、统计年鉴 | 产量、价格、行业规模、区域经济 | 注意统计口径和更新时间 |
| 中国司法风险 | 中国执行信息公开网、法院公告、裁判文书公开渠道 | 被执行、失信、诉讼、司法拍卖线索 | 覆盖和公开程度可能变化 |
| 中国知识产权 | 国家知识产权局及专利检索系统 | 专利、商标、技术布局 | 需区分申请、授权、有效状态 |
| 中国招投标 | 中国政府采购网、各省公共资源交易平台、行业招投标平台 | 客户、订单、价格、竞争者 | 中标金额和履约收入不是同一概念 |
| 美国上市公司可比 | SEC EDGAR APIs | 10-K、10-Q、8-K、XBRL 财务数据 | 适合可比公司和行业披露 |
| 全球宏观 | World Bank Indicators API、IMF DataMapper | GDP、人口、通胀、贸易、宏观变量 | 适合跨境项目背景 |
| 利率和宏观时间序列 | FRED | 利率、通胀、汇率、信用利差 | API key 规则需按官网确认 |

## 找不到免费权威来源时

不得用低质量来源强行填补。应在报告中写清：

- 未发现可免费获取的权威公开来源。
- 当前可用证据等级和局限性。
- 需要线下获取的材料：合同、银行流水、审计报告、客户访谈、供应商访谈、监管批文、现场尽调。
- 可选付费或授权数据源：企业征信、行业数据库、券商数据库、PitchBook、Wind、Choice、Capital IQ、FactSet 等。
- 对投资判断的影响：是否影响可投结论、估值、收益率、CP 条件或退出路径。
