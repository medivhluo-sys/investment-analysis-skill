# 行业数据采集提示

本文件用于第一层前线采数。目标是按行业提醒前线人员优先收集什么，不做机构级行业适配器。

## 使用规则

- 先识别标的主要收入来源所在行业。
- 若同时跨多个行业，按收入、利润或估值贡献最大的业务线优先。
- 表中“公开可查”用于公开核验，“公司必须提供”用于资料请求，“关键风险信号”用于访谈和红旗登记。

## 行业提示表

| 行业 | 公开可查 | 公司必须提供 | 关键风险信号 |
|------|----------|--------------|--------------|
| 制造业/工业品 | 行业产能、产量统计、原材料价格、可比公司毛利、环保/安全处罚 | 设备清单、产能、产量、良率、库存、前五大客户、前五大供应商、质量记录 | 产能利用率与收入不匹配；库存增长快于订单；毛利远高于同行 |
| SaaS/软件 | 竞品定价、软著、专利、招投标、客户案例、可比公司 ARR 口径 | ARR/MRR、续费率、流失率、CAC、LTV、回款、递延收入、实施收入占比 | 一次性实施收入占比过高；ARR 包含未签约 pipeline；增长不转化为回款 |
| 医疗器械/医药 | 注册证/备案、集采政策、药监处罚、医保和招采信息、可比公司公告 | 注册证、医院准入、经销商库存、终端销售、回款周期、质量体系、临床或检测资料 | 注册证过期或不覆盖核心产品；渠道库存高；销售依赖不合规推广 |
| 工程项目/EPC/环保系统 | 招投标、中标公告、政府采购、可比公司项目毛利、客户信用 | 合同、开工、里程碑、验收、发票、回款、应收账龄、质保金、项目成本 | 利润确认早于验收；应收账款和质保金过高；项目毛利波动异常 |
| 消费/渠道 | 电商销售、门店信息、价格体系、竞品、舆情、食品/产品安全处罚 | SKU、渠道结构、sell-in/sell-through、退货、返点、库存、复购率、广告投放 | 渠道压货；退货和返点未扣减收入；获客成本过高 |
| 硬科技/半导体 | 专利、标准、客户认证新闻、上市可比公司、出口管制或产业政策 | 样品/量产阶段、设计导入、良率、测试报告、IP 权属、客户认证、供应链 | 国产替代叙事无客户验证；库存堆积；核心 IP 不在标的 |
| 资产/能源/基础设施 | 资产权属、许可、环评、安全、价格政策、补贴、并网/特许经营信息 | 权属文件、许可、合同现金流、运维成本、负债、保险、维修资本开支 | 权属或许可不完整；现金流依赖弱信用方；维护资本开支被低估 |
