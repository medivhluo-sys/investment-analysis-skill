# 前线数据采集清单

本文件给前线人员使用。目标是让非专业尽调人员知道第一次接触、拿到 BP、访谈和现场观察时应收集什么。每项数据都要记录来源、日期、提供人和是否原件。

## 使用方式

1. 先完成 `preflight-gate.md` 的门禁判断。
2. 按本清单向标的公司索取资料。
3. 拿不到理想资料时，使用 `alternative-data-sources.md` 的次优资料或最低替代。
4. 每个空白项都要写明影响哪个投资判断。

## 一、第一次接触前可自行查询

| 数据 | 动作 | 用途 |
|------|------|------|
| 公司主体 | 查询国家企业信用信息公示系统或同等级官方登记入口 | 确认标的是否存在、经营状态、股东和变更线索 |
| 司法风险 | 查询执行、失信、裁判文书、法院公告等公开入口 | 识别诉讼、执行、担保和重大争议线索 |
| 行政处罚 | 查询市场监管、税务、生态环境、应急管理、药监等主管部门 | 识别合规和停产风险 |
| 知识产权 | 查询专利、商标、软著、许可和权属 | 验证技术、品牌和产品壁垒 |
| 招投标和客户线索 | 查询政府采购、公共资源交易、企业官网和新闻 | 验证客户、订单、项目和价格线索 |
| 可比公司 | 查找上市公司公告、招股书、年报、债券募集说明书 | 获取行业、毛利、费用、估值和风险对标 |

## 二、拿到 BP 后向公司索取

### 主体和股权

| 必要资料 | 用途 |
|----------|------|
| 营业执照、公司章程、股东名册、最新工商档案 | 确认投资主体、历史沿革和权利基础 |
| 投前和投后股权结构图 | 确认实控人、稀释、平台和关联方 |
| 历次融资、股权转让、增资协议或决议 | 识别估值变化、特殊权利和潜在纠纷 |
| 董事会、监事、保留事项、信息权安排 | 判断治理和投资人保护 |

### 财务和现金

| 必要资料 | 用途 |
|----------|------|
| 近三年审计报告或财务报表、最近一期管理报表 | 判断收入、利润、资产负债和趋势 |
| 收入明细，按客户、产品、区域、渠道拆分 | 验证增长质量和集中度 |
| 应收账款账龄、回款记录、银行流水或月度余额 | 验证收入是否转化为现金 |
| 成本明细、供应商明细、库存和固定资产清单 | 验证毛利、产能和资产真实性 |
| 税务申报、发票、贷款、担保、质押和或有负债资料 | 识别税务、债务和表外风险 |

### 业务和经营

| 必要资料 | 用途 |
|----------|------|
| 产品清单、价格表、产品阶段和技术参数 | 判断公司实际卖什么 |
| 前十大客户清单、合同、订单、交付、验收、发票和回款 | 穿透收入真实性 |
| 前十大供应商清单、采购合同、入库单和付款记录 | 验证成本和供应链稳定性 |
| 产能、产量、良率、稼动率、库存、交付周期 | 验证经营数据是否支撑财务数据 |
| 资质、许可、认证、环保、安全、质量体系文件 | 判断是否存在投产或经营前置条件 |

### 交易条款

| 必要资料 | 用途 |
|----------|------|
| 投资金额、估值、股权比例、工具类型 | 确认本次交易的经济实质 |
| 资金用途、付款安排、交割条件 | 判断交易可执行性和资金风险 |
| 回购、差补、对赌、优先清算、反稀释、退出权 | 判断下行保护和真实收益来源 |
| 退出路径说明、可比退出案例、潜在买方或上市规划 | 判断收益兑现路径 |

## 三、访谈必问问题

### 创始人或负责人

- 这个生意为什么现在成立，最关键的外部变化是什么？
- 公司真正不可替代的能力是什么，证据是什么？
- 未来 12-24 个月最可能出错的三个假设是什么？
- 如果融资不到位，公司如何调整计划？
- 投资人最应该担心但 BP 没写清楚的问题是什么？

### 财务负责人

- 收入确认政策是什么，哪些收入尚未验收或回款？
- 前五大客户的账期、逾期和后续回款情况如何？
- 毛利率变化由价格、成本、产品结构还是一次性因素驱动？
- 未来增长需要多少营运资金和资本开支？

### 销售或客户负责人

- 客户从线索到合同、交付、回款的周期多长？
- 最近丢单的原因是什么？
- 前五大客户是否可访谈，是否有续约或扩单证据？
- 是否存在渠道压货、返点、退货或价格保护？

### 产品、技术或运营负责人

- 产品处于样品、试产、量产、交付还是稳定运营阶段？
- 关键技术或工艺是否自有，IP 属于谁？
- 产能、良率、质量、售后和库存是否与销售计划匹配？
- 哪些环节外包，关键供应商是否可替代？

## 四、现场观察

| 场景 | 看什么 |
|------|--------|
| 办公场地 | 团队规模、岗位结构、财务和销售是否真实运转 |
| 工厂或仓库 | 设备、产线、库存、原材料、成品、出入库记录 |
| 项目现场 | 建设进度、验收节点、实际使用情况、客户或监管标识 |
| 研发环境 | 样品、测试记录、专利或软著对应的实际产品 |

现场观察只能作为线索，不能替代合同、回款、验收、监管许可等证据。
