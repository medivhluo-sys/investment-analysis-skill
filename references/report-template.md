# 投资分析报告结构模板

## 前端（投资人打开即见）

1. 封面 — 标的 + 日期 + 素材来源
2. 术语表 — 所有专业名词首次出现需解释
3. KPI 仪表盘 — 6 个核心指标，固定列数
4. **总体结论与重要提示** — 最简短的判断 + 决策标准表 + 收益计算逻辑
5. **关于本报告** — 素材基础 / 额外工作 / 核心假设 / 局限性 / 标记体系
6. **核心观点与后续行动** — CP 条件 + 假设表 + 三段式工作建议

## 正文（前结论后展开）

7. 项目概览 — 2-3 句 + 双栏亮点/风险
8. 投资方案 — 含收益率测算方法说明
9. 份额回报测算 — 含情景独立性验证
10. 尽调清单速查 — 标注可联网 vs 需线下
11-19. 各维度详细分析

## 后端

20. 附录：独立交叉核验摘要
21. 附录：参考来源与引注
22. 附录：PPT 素材全量验证表

## 决策标准表（必须包含）

以投资人设定的最低年化收益率为基准：

| 关键变量 | 可投区间 | 当前值 | 判断 |
|---------|---------|--------|------|
| （核心定价变量） | ≥ XX | 当前市场价 | ✅/⚠️/❌ |
| （核心成本变量） | ≤ XX | 当前市场价 | ✅/⚠️/❌ |
| （产能/利用率） | ≥ XX% | 预计值 | ✅/⚠️/❌ |
| （关键前置条件） | 已满足 | 当前状态 | ✅/⚠️/❌ |

## 后续工作三段式

### 一、后续尽调（投资决策前）
- 按周排期，每项标注目的（验证哪条 CP 条件）
- 区分「可联网验证」vs「需线下获取」

### 二、投资协议关键条款
- CP 条件 / 差补保护线 / 创始人锁定 / 信息权 / 反稀释 / 退出权

### 三、投后管理
- 分阶段：建设期 → 爬坡期 → 稳定运营 → 退出准备
- 每阶段含：频率 + 核心关注点 + 预警信号

## 格式规范

- 标题不用 emoji
- 正文 Georgia 衬线 + 中文宋体，标题 sans-serif
- 段落 3 句以上拆 bullet
- 表格表头大写，列宽手动指定
- callout 用全边框不用左侧彩色条
- KPI 卡片固定列数，数字带 `white-space: nowrap`
- PDF 用 Playwright，`display_header_footer=False`
